# Instalasi

1. Unduh Node JS (minimal v16.20.0)
2. Jalankan perintah berikut pada terminal untuk menginstal dependensi:
   ```
   npm install
   ```
3. Jalankan perintah berikut untuk menjalankan aplikasi:
   ```
   npm run dev
   ```

## Tech stack yang digunakan

### 1. **Framework**

- React.js

### 2. **Styling Tool**

- Material UI (MUI)
- Tabler Icons

### 3. **Version Control**

- Git (GitHub)

### 4. **Build Tool**

- Vite
