# Instalasi

1. Unduh Node JS (minimal v16.20.0)
2. Jalankan perintah berikut pada terminal untuk menginstal dependensi:
   ```
   npm install
   ```
3. Jalankan perintah berikut untuk menjalankan aplikasi:
   ```
   npm run dev
   ```
